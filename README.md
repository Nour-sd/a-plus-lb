# A Plus lb

A Plus lb is a modern educational platform built for students and teachers in Lebanon.  
It helps students find online tutoring, agenda help, courses, SAT preparation, Arabic and English support, web development courses, Microsoft Office support, and guided project help.

## Features

- Student and teacher dashboards
- Teacher application system
- Course browsing and application flow
- Admin approval workflow
- Chat system concept
- Course material uploads
- Payment confirmation through Whishmoney / OMT
- Responsive design for desktop and mobile

## Technologies Used

- HTML
- CSS
- JavaScript
- Firebase
- Firestore
- Firebase Hosting
- Responsive Web Design
